# Timmyovo Local

## How do I install these formulae?

`brew install timmyovo/local/<formula>`

Or `brew tap timmyovo/local` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "timmyovo/local"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
